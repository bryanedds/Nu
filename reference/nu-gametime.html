GameTime Type

Namespace: Nu
Assembly: Nu.dll
Base Type: ValueType
All Interfaces: IComparable , IComparable<GameTime> , IEquatable<GameTime>
Kind: Struct

Provides a variable representation of time based on whether the engine is configured to use a static or a dynamic frame rate.
